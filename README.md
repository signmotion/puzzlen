Заготовка для игры "пятнашки".

Запускается приложение из консоли командой "puzzlen [N [M]]".
Где N, M - количество ячеек по ширине и высоте.
Пример: puzzlen 7 10

Управление
  LeftClick + move  Перемещает элемент.
  SPACE             Перетасовывает элементы.
  ESC               Выход.
